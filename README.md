# Breast Cancer Prediction using Machine Learning

## Overview

This project was developed as part of my learning journey in Machine Learning. The objective is to predict whether a breast tumor is **Benign** or **Malignant** using a Logistic Regression model. The project covers the complete machine learning workflow, including data preprocessing, exploratory data analysis, model training, evaluation, and prediction.

---

## Features

- Data preprocessing and cleaning
- Exploratory Data Analysis (EDA)
- Feature scaling using StandardScaler
- Logistic Regression model implementation
- Model evaluation using accuracy and classification metrics
- Prediction on unseen data

## Machine Learning Workflow

1. Import the required libraries.
2. Load and inspect the dataset.
3. Perform data preprocessing.
4. Analyze the dataset using visualizations.
5. Split the data into training and testing sets.
6. Apply feature scaling using StandardScaler.
7. Train the Logistic Regression model.
8. Evaluate the model using performance metrics.
9. Make predictions on new data.
